## ✅ 設定1: LIFFアプリのエンドポイントURL

### 何を設定？


### 手順（画像付きで説明）
1. https://developers.line.biz/console/ を開く
2. チャネルを選択
3. 左メニューの「**LIFF**」をクリック
4. LIFFアプリ `2008962886-K2CRmPwV` をクリック
5. 「**エンドポイントURL**」の欄に上記URLを貼り付け
6. 「**更新**」ボタンをクリック

## ✅ 設定2: Webhook URL

### どこで設定？
LINE Developers Console → チャネル → **Messaging API**タブ

### 何を設定？
```
https://unadministrable-idiocratically-annalee.ngrok-free.dev/api/line/webhook
```

### 手順
1. https://developers.line.biz/console/ を開く
2. チャネルを選択
3. 左メニューの「**Messaging API**」をクリック
4. 下にスクロールして「**Webhook URL**」セクションを見つける
5. 「**Webhook URL**」の欄に上記URLを貼り付け
6. 「**検証**」ボタンをクリック
   - ✅ 「検証に成功しました」と表示されればOK
7. 「**Webhookの利用**」を「**利用する**」に変更
8. 「**更新**」ボタンをクリック

## 🧪 動作確認

- 自動的にメッセージが返ってくればOK ✅

### 2. LIFFアプリの確認
- LINEアプリで「**メニュー**」と送信
- LIFFアプリが開けばOK ✅

## ⚠️ 注意事項

**ngrokを再起動するとURLが変わります！**

URLが変わったら：
1. `./get-ngrok-url.sh` で新しいURLを確認
2. 上記2つの設定を新しいURLに更新
